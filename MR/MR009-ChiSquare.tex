\begin{frame}
\frametitle{Chi Square Test = Computing the critical value}
The degrees of freedom ( df or $\eta$ ) is computed as

\[ \eta = (r-1)\times(c-1) \]

\begin{itemize}
\item $r$ number of rows
\item $c$ number of columns
\end{itemize}
